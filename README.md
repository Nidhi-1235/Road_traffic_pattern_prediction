# Mini Project

## Overview
This project is a Python-based web application developed using machine learning and web technologies.

## Technologies Used
- Python
- Flask
- Machine Learning
- HTML
- CSS

## Project Structure
- app.py - Main application file
- model.py - Machine learning model
- templates - HTML pages
- static - CSS and assets
- demo_data - Dataset files

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

python app.py
